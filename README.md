

# AI Puzzle Mobile Game

This is an AI-based 8 Puzzle mobile game developed using Flutter and Python Flask.

## Features
- Solvable shuffled puzzles
- Hint system
- Auto solver
- Move counter
- Timer
- Win detection

## Technologies Used
- Flutter
- Dart
- Python Flask
- AI Search Algorithm
